How To Run Program:
1) Download, and extract .zip file.
2) Navigate inside the extracted folder.
3) Run main file using:
python main.py

This main file will run Tester.py and demonstrate what we have implemented in our sudoku solver program. 
***This class contains the main() method.***

Sudoku.py is our base sudoku class which handles all encoding of the puzzle, and running our DPLL method.

DPLL.py is our implemented DPLL algorithm which checks a set of input for SAT.

sudoku1-9 are our puzzle input files, providing the solver varying difficulties of puzzles, and different blank spaces to be filled in.
